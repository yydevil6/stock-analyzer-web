# A 股本地网页版半自动盯盘助手

一个使用 Python + Flask、HTML、CSS 和 JavaScript 构建的本地网页工具。它用于查看或手动录入 A 股行情，管理自选股，计算持仓盈亏和超短线评分，并在盘中给出价格提醒与操作计划。

> 本工具仅供学习和辅助分析，不构成投资建议。它不连接券商、不自动交易、不自动买卖、不自动下单，也不会保存股票账户、券商账户、密码或 Cookie。

## 功能

- 获取股票名称、当前价、今日高低价、涨跌幅、成交额和量比
- 真实行情失败时使用模拟数据兜底，并保留完整手动输入能力
- 手动输入行情、成本价、持仓股数和三个提醒价
- 计算当前市值、浮动盈亏、价格差距和三个参考位
- 生成 0–100 分超短线评分、当前状态和今日操作计划
- 使用浏览器 `localStorage` 保存自选股和提醒记录，无需数据库
- 自选股支持获取行情、分析、编辑和删除
- 支持 30、60、120 秒自动刷新，可随时停止
- 支持止损、冲高卖出、强压力提醒，并设置 5 分钟防重复提醒
- 电脑和手机响应式布局

## 行情与股票名称

输入 6 位 A 股代码并点击“获取行情”。后端会尝试读取无需登录或密钥的腾讯公开行情数据，并统一返回：

- `code`
- `name`
- `current_price`
- `high`
- `low`
- `change_percent`
- `amount`
- `volume_ratio`

股票名称优先使用接口返回的中文名称。接口缺少名称时使用本地映射，映射包含航天发展、东方海洋、平安银行、贵州茅台、宁德时代、比亚迪、中国平安、招商银行、工商银行和五粮液。仍无法识别时显示“未知股票名称”，不会用代码冒充名称。

如果真实接口失败，页面显示“行情获取失败，已切换为手动输入模式”。页面不会空白，用户可以继续修改任何字段并分析。

## 超短线评分

评分从 50 分开始，最终限制在 0–100 分：

- 当前价高于成本：+20
- 接近今日高点：+15
- 今日涨跌幅为正：+15
- 量比大于 1：+10
- 远离今日低点：+10
- 当前价低于成本：-20
- 接近今日低点：-15
- 今日涨跌幅为负：-15
- 量比小于 0.8：-10
- 跌破止损提醒价：-30

状态与计划：

- 75–100：偏强，持有观察，冲高分批止盈
- 50–74：震荡，不追高，接近压力位减仓
- 0–49：偏弱，跌破止损位要控制风险
- 触发止损提醒时优先显示“触发止损”
- 触发冲高卖出提醒时优先显示“接近卖出位”

## 自选股

1. 填写股票代码、名称、成本价、持仓股数和提醒价。
2. 点击“保存到自选股”。
3. 自选股卡片会显示当前价、涨跌幅、成本、股数、市值、盈亏、评分、状态、计划和最后更新时间。
4. “获取行情”只更新这一只股票；“分析”回填并重新计算；“编辑”只回填表单；“删除”移除该股票。

数据保存在当前浏览器的 `localStorage`。刷新页面后仍然存在；清除浏览器网站数据后会被删除。

## 自动刷新

1. 先保存至少一只自选股。
2. 选择 30、60 或 120 秒。
3. 点击“开启自动刷新”。
4. 页面显示开启状态和最后更新时间。
5. 再次点击“停止自动刷新”即可停止。

自动刷新只读取行情并更新分析结果。真实行情失败时会保留已有数据并显示友好提示。

## 提醒价与记录

- 当前价小于或等于止损提醒价：显示红色风险提醒
- 当前价大于或等于冲高卖出提醒价：显示绿色止盈提醒
- 当前价大于或等于强压力提醒价：显示橙色回落风险提醒

提醒记录包含时间、代码、名称、当前价、类型和内容，最新记录排在最上方。记录保存在 `localStorage`，并可点击“清空提醒记录”删除。相同股票、相同提醒在 5 分钟内不会重复记录。

## Windows 运行方法

先安装 Python 3.10 或更高版本，并在安装时勾选 **Add Python to PATH**。

在项目目录打开 PowerShell，安装依赖：

```powershell
py -m pip install -r requirements.txt
```

启动网页：

```powershell
py app.py
```

浏览器打开：

<http://127.0.0.1:5000>

停止服务时在 PowerShell 按 `Ctrl + C`。

## 完整功能测试

1. 输入 `000547` 获取行情，确认显示“航天发展”。
2. 输入 `000001` 获取行情，确认显示“平安银行”。
3. 断开网络后再获取行情，确认页面不空白并可手动输入。
4. 填写行情、成本和股数，点击“开始分析”，检查市值、盈亏、评分、参考位和计划。
5. 填写三个提醒价并保存自选股，刷新页面确认数据仍在。
6. 点击自选股的获取行情、分析和编辑按钮，确认功能正常。
7. 将止损提醒价设置为高于当前价，重新保存并分析，确认出现红色止损提醒和提醒记录。
8. 将冲高卖出提醒价设置为低于当前价，确认出现绿色止盈提醒。
9. 开启自动刷新，检查状态和最后更新时间，然后停止。
10. 测试清空提醒记录和删除自选股。

## API

- `GET /api/quote/<股票代码>`：获取真实行情或模拟兜底数据
- `POST /api/analyze`：根据行情、持仓和提醒价生成分析结果

项目只提供行情查看、辅助分析、提醒和本地记录，不执行任何交易操作。
