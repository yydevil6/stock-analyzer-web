# 超短线自选股分析助手

一个基于 Python + Flask 的盘中盯盘提醒工具。支持获取或手动录入行情、保存自选股、定时刷新、价格提醒、持仓盈亏、超短线评分与今日操作计划。

> 本项目只做手动数据分析，不连接券商、不自动交易、不自动下单，也不构成投资建议。

## 功能

- 手动输入股票代码、名称、当前价、日内高低价、涨跌幅、成交额与量比
- 输入买入成本与持仓股数，保留第二版持仓分析能力
- 自动计算当前市值、浮动盈亏金额和浮动盈亏比例
- 计算距离成本价、今日最高价与今日最低价的差距
- 生成 0–100 分超短线强弱评分、止损位、卖出参考位与今日计划
- 使用浏览器 `localStorage` 保存自选股，无需数据库或新增依赖
- 自选股支持刷新后保留、自动回填分析、更新最近价格和盈亏、删除
- 输入股票代码后可获取公开行情，并自动填入名称、价格、涨跌幅、成交额和量比
- 真实行情超时或解析失败时自动使用稳定模拟数据兜底，并保留手动输入
- 支持 30、60、120 秒自动刷新自选股行情，可随时开启或停止
- 每只自选股可设置止损、冲高卖出和强压力三个提醒价
- 盯盘卡片显示行情、持仓、市值、盈亏、评分、状态和操作计划
- 提醒记录保存在 `localStorage`，刷新后仍然存在，并可一键清空
- 响应式页面，支持电脑和手机浏览

## 获取行情

输入 6 位 A 股股票代码并点击“获取行情”，Flask 后端会尝试从腾讯公开行情端点读取数据。该端点无需登录或密钥，项目也不会保存任何账号密码。请求超时、网络不可用或数据格式异常时，接口会返回模拟行情兜底，页面同时提示“行情获取失败，请稍后重试或手动输入”，用户可以继续修改任何字段。

行情获取只会填充表单，不会自动保存自选股、自动分析、连接券商或执行交易。

行情接口会统一返回 `code`、`name`、`current_price`、`high`、`low`、`change_percent`、`amount` 和 `volume_ratio`。股票名称优先采用行情源返回的中文名称；若名称缺失或错误地等于股票代码，则使用内置名称映射。目前包含：

- `000547`：航天发展
- `002086`：东方海洋
- `000001`：平安银行
- `600519`：贵州茅台
- `300750`：宁德时代

不在映射中的股票若也没有有效中文名称，将显示“未知股票名称”，不会用股票代码代替名称。

## 自选股保存

填写股票代码、股票名称、买入成本和持仓股数后，点击“保存到自选股”即可保存。若行情字段也已填写，会一并保存为最近的行情快照。点击列表中的“分析”会自动回填数据并执行分析；分析成功后，列表中的最近当前价和浮动盈亏会自动更新。

自选股数据只保存在当前浏览器的 `localStorage` 中，刷新页面仍会保留。清除浏览器网站数据后，自选股也会被清除。

## 盘中盯盘提醒

保存自选股前可以填写三个可选提醒价；留空时系统会根据成本价和最近价格生成默认值：

- 止损提醒价：当前价小于或等于该价格时显示风险提醒
- 冲高卖出提醒价：当前价大于或等于该价格时显示分批止盈提醒
- 强压力提醒价：当前价大于或等于该价格时提示回落风险

在“我的自选股列表”中选择 30、60 或 120 秒，点击“开启自动刷新”。系统会只读获取每只自选股的行情并重新计算；再次点击即可停止。真实行情失败时不会覆盖已有盯盘数据，也不会导致页面空白。

提醒采用状态变化记录方式：同一提醒持续触发时不会在每次刷新中重复写入；价格离开阈值后再次触发，才会新增记录。提醒和自选股都只保存在当前浏览器。

## 超短线评分规则

评分以 50 分为基准：当前价高于成本、接近今日高点、量比大于 1、涨跌幅为正会加分；跌破成本、接近今日低点或涨跌幅为负会扣分，最终限制在 0–100 分。

- 75–100：偏强，持有观察，冲高分批止盈
- 50–74：震荡，不追高，接近压力位减仓
- 0–49：偏弱，跌破止损位要控制风险

## Windows 运行方法

### 1. 安装 Python

从 [Python 官网](https://www.python.org/downloads/) 安装 Python 3.10 或更高版本。安装时请勾选 **Add Python to PATH**。

### 2. 打开项目目录

在项目文件夹空白处按住 `Shift` 并点击鼠标右键，选择“在终端中打开”，或者在 PowerShell 中进入项目目录：

```powershell
cd stock-analyzer-web
```

### 3. 创建并启用虚拟环境

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

如果 PowerShell 阻止运行激活脚本，可先执行：

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\.venv\Scripts\Activate.ps1
```

### 4. 安装依赖

```powershell
python -m pip install -r requirements.txt
```

### 5. 启动项目

```powershell
python app.py
```

浏览器访问：<http://127.0.0.1:5000>

停止服务时，在终端按 `Ctrl + C`。

## 第六版快速测试

1. 输入 `000547`，点击“获取行情”，确认名称显示“航天发展”。
2. 填写成本价、持仓股数和三个提醒价，点击“保存到自选股”。
3. 点击“开始分析”，确认盯盘卡片显示市值、盈亏、评分、状态和计划。
4. 把止损提醒价设为高于当前价，重新保存并分析，确认出现“已触发止损提醒，请控制风险”。
5. 刷新页面，确认自选股和“今日提醒记录”仍然存在。
6. 选择刷新间隔，点击“开启自动刷新”，确认按钮变为“停止自动刷新”并显示最后更新时间；再次点击停止。
7. 可使用“清空提醒记录”删除本地提醒历史。

## API 示例

`POST /api/analyze`

```json
{
  "stock_code": "000547",
  "stock_name": "航天发展",
  "current_price": 20.05,
  "high_price": 20.60,
  "low_price": 19.20,
  "change_percent": 3.20,
  "turnover": 12.50,
  "volume_ratio": 1.35,
  "cost_price": 18.50,
  "shares": 1000
}
```

服务会返回 JSON 格式的自选股超短线分析报告。行情可以由用户手动输入或通过只读行情接口填充；系统不会连接券商或执行任何交易操作。
